----------------------------------------------------------------------------------------------------------------------


文件名:
square_random.m
功能:
在正方形区域内随机均匀分布节点
定义的函数:
function square_random(square_L,nodes_n,anchors_n,GPS_error)
参数说明: 
square_L: 正方形区域的边长
node_n: 节点数目
anchors_n: 锚节点数目. 小于1表示比例,大于1表示数目
GPS_error:锚节点又GPS产生的位置最大误差，GPS获得的位置到真实位置的距离最大偏差
调用格式:
square_random(200,200,0.1)
square_random(200,200,20)

_____________________________________________________________________

文件名:
square_regular.m
功能:
在正方形区域内网格状分布节点
定义的函数:
function square_regular(square_L,grid_L,deploy_error,anchors_n,GPS_error)
参数说明: 
square_L: 正方形区域的边长
grid_L: 网格边长
deploy_error:部署误差是网格边长的多少;为0表示点正好分布在网格点上;这个参数应该小于0.5
anchors_n: 锚节点数目. 小于1表示比例,大于1表示数目
GPS_error:锚节点又GPS产生的位置最大误差，GPS获得的位置到真实位置的距离最大偏差
调用格式:
square_regular(200,20,0,0.1)
square_regular(200,20,0,12)

_____________________________________________________________________

文件名:
C_random.m
功能:
在C型区域内随机均匀分布节点
定义的函数:
function C_random(area,nodes_n,anchors_n,GPS_error)
参数说明: 
area: 描述C型区域,1x4的数组[L,X,Y1,Y2]
|----------------------------------------------------------------------------------------------------|
|				L			|
|							|
|							| L
|							|
|				Y2			|
|		|-----------------------------------------------------------------------|
|		|
|		|
|		|
|		|
|		|
|		|
|		|  
| 0		|  X
|		|
|		|
|		|
|		|
|		|
|		|-----------------------------------------------------------------------|
|				Y1			|
|							|
|							| L
|							|
|				0			|
|----------------------------------------------------------------------------------------------------|
nodes_n: 节点数目
anchors_n: 锚节点数目. 小于1表示比例,大于1表示数目
GPS_error:锚节点又GPS产生的位置最大误差，GPS获得的位置到真实位置的距离最大偏差
调用格式:
C_random([200 40 40 160],150,0.1)
C_random([200 40 40 160],150,15)

文件名:
C_regular.m

在C型区域内网格状分布节点
定义的函数:
function C_regular(area,grid_L,deploy_error,anchors_n,GPS_error)
参数说明: 
area:描述C型区域

____________________________________________________________________

文件名:
coordinates.mat
功能:
该数据文件保存了一个all_nodes结构体
上面四个文件定义的节点部署函数都将得到的所有节点坐标以及相应的场景参数保存在all_nodes结构体中

____________________________________________________________________

文件名:
Distribution_Of_WSN.m
功能:
根据coordinates.mat画出节点分布图(Figures文件夹中已经保存了一些图）
调用格式:先调用节点部署函数,然后再查看其节点分布
square_random(200,200,0.1)
Distribution_Of_WSN
----------------------------------------
square_regular(200,20,0,0.1)
Distribution_Of_WSN
----------------------------------------
C_random([200 40 40 160],150,0.1)
Distribution_Of_WSN
----------------------------------------
C_regular([200 40 40 160],10,0,0.1)
